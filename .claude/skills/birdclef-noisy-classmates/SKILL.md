## Why this beats vanilla Noisy Student

| Vanilla Noisy Student | Noisy Classmates |
|---|---|
| 1 teacher → 1 student, **iterative** (round 1 → 2 → 3 …) | K classmates train **simultaneously** in one loop |
| Student learns from teacher's own pseudo-labels → confirmation bias compounds | Each classmate learns from **other classmates'** pseudo-labels — no self-loop |
| Same architecture → correlated errors | **Different architectures** → decorrelated errors → ensemble diversity |
| Pseudo labels fixed per round | Pseudo labels **refresh every step** as classmates evolve |

## Algorithm (reconstructed)

```python
# K = number of classmate models (typically 3-4, e.g. CNN + Transformer + SSM + ViT)
# D_L = labeled data (train_audio + train_soundscapes_labels.csv windows + supplemental XC/iNat)
# D_U = unlabeled data (train_soundscapes minus labeled windows)
# strong_aug = SpecAugment + Mixup + Gaussian noise + time-shift
# weak_aug   = light augmentation or none

# Init: K classmates with diverse architectures
classmates = [build_model(arch_k) for k in range(K)]
optimizers = [build_optimizer(m) for m in classmates]

for step in range(num_steps):
    # Labeled batch (clean signal)
    x_l, y_l = sample_labeled(D_L)

    # Unlabeled batch (where the magic happens)
    x_u = sample_unlabeled(D_U)
    x_u_weak   = weak_aug(x_u)    # what classmates predict on to generate pseudo labels
    x_u_strong = strong_aug(x_u)  # what classmates are trained on

    # Co-evolutionary step: each classmate predicts on weak-aug, trained on strong-aug
    with torch.no_grad():
        pseudo_logits = [m(x_u_weak) for m in classmates]   # K tensors of (B, C)
        pseudo_probs  = [torch.sigmoid(z) for z in pseudo_logits]

    for k, model in enumerate(classmates):
        # Supervised loss on labeled data
        loss_sup = bce_with_logits(model(strong_aug(x_l)), y_l)

        # Pseudo-label target = ensemble of OTHER classmates (key: exclude self!)
        teachers = [p for j, p in enumerate(pseudo_probs) if j != k]
        pseudo_target = sharpen(mean(teachers), T=0.5)
        mask = (pseudo_target.max(dim=-1).values > tau)       # confidence gating

        # Unsupervised consistency loss
        student_logits_u = model(x_u_strong)
        loss_unsup = (bce_with_logits(student_logits_u, pseudo_target) * mask).mean()

        loss = loss_sup + lambda_u * loss_unsup
        optimizers[k].zero_grad(); loss.backward(); optimizers[k].step()
```

### Key design choices

1. **K = 3 or 4** classmates. Diminishing returns past 4.
2. **Architecture diversity is mandatory**: CNN + Transformer + SSM (e.g. EfficientNet-B0 + PVT-Tiny + Proto-SSM). Two copies of the same architecture defeats the purpose.
3. **`teachers = [p for j, p in enumerate(pseudo_probs) if j != k]`** is the anti-confirmation-bias trick. Self-distillation is explicitly excluded.
4. **Sharpening temperature** `T ≈ 0.5` makes pseudo-targets crisper (helps when ensemble agrees).
5. **Confidence mask** with threshold `τ ≈ 0.7` for high-confidence windows only — early in training most windows are dropped, late in training most pass through.
6. **`lambda_u` ramp-up**: 0 → 1.0 over first ~10% of steps so classmates first learn supervised signal, then start absorbing pseudo-labels.
7. **Strong aug on labeled too** (not just unlabeled): "noisy classmate" means the student always sees perturbed input, regardless of label availability.

## Concrete defaults for BirdCLEF+ 2026

### Input representation

- All classmates consume **Perch v2 embeddings** at 5-sec windows, 1536-dim, 12 windows per 60-sec soundscape (= 32 kHz raw audio in cache: `external/kaggle_kernels/eos5/datasets/tuckerarrants__birdclef-2026-waveform-cache/`)
- Embeddings are **cached once** to disk (~30-60 min on A100). After that every epoch is embedding-only.

### Architecture pool (start with 3)

| Branch | Architecture | Inductive bias |
|---|---|---|
| A | **Proto-SSM** (existing, [experiments/single_proto_ssm/](../../../experiments/single_proto_ssm/)) | Bidirectional SSM + prototypes — temporal continuity |
| B | **MLP-Mixer head** over (12, 1536) | Per-window features + cross-window mixing — Transformer-like without attention cost |
| C | **PVT-Tiny adapter** — project (12, 1536) into PVT-tiny patch embedding, predict 234 logits | Spatial reasoning over time-feature plane |
| (D) | EfficientNet-B0 1D head if 4th branch wanted | Convolutional locality |

### Loss

```
L_k = BCE_with_logits(f_k(strong_aug(x_l)), y_l)
    + λ_u(step) · masked_BCE(f_k(strong_aug(x_u)), sharpen(mean_j≠k σ(f_j(weak_aug(x_u))), T=0.5))
```

### Augmentation (embedding-level, fast)

- **Mixup** on embeddings α = 0.4
- **Gaussian noise** σ = 0.05 × per-dim std
- **Random feature dropout** p = 0.1
- **Window order shuffle** with probability 0.3 (BirdCLEF-specific: bird calls are stationary in time, so shuffle is meaningful)

Audio-level augmentation (SpecAugment etc.) only if re-Perching is acceptable.

### Hyperparameters (starting point)

| Knob | Value |
|---|---|
| K classmates | 3 |
| Batch size labeled | 64 |
| Batch size unlabeled | 192 (3× labeled) |
| Optimizer | AdamW, weight_decay=0.01 |
| LR | 8e-4, cosine annealing |
| Warmup | 5% of steps |
| λ_u ramp | 0 → 1 over first 10% steps |
| Sharpen T | 0.5 |
| Confidence mask τ | 0.7 |
| Steps | 30 epochs over labeled set |
| Half precision | bf16 on A100 |

### Data sources to use

1. `birdclef-2026/train_audio/` — 35,549 files, 234 classes, single primary_label (+ optional secondary_labels)
2. `birdclef-2026/train_soundscapes_labels.csv` — 1,478 labeled windows (66 files), multi-label
3. `birdclef-2026/train_soundscapes/` (unlabeled portion ≈ 10K files × 12 windows) — **fuel for self-training**
4. Supplemental `external/xc_supplement/` (43 files) + `external/inat_supplement/` (276 files) — add to D_L for scarce species

## Failure modes to watch

| Symptom | Likely cause | Fix |
|---|---|---|
| All classmates converge to same predictions | λ_u too high or architectures too similar | Lower λ_u, swap in more diverse arch |
| Pseudo-label distribution collapses to dominant species | Confirmation bias snuck in (self in own teacher set?) | Audit teacher index loop |
| OOF AUC drops below single-model baseline | Strong aug too aggressive early; supervised signal lost | Delay λ_u ramp until step 20% |
| Insecta class accuracy unchanged | Sonotype labels don't have meaningful pseudo-label signal | Mask out 47158son* from unsup loss |
| GPU OOM | Storing 4 models + 4 graphs | Gradient accumulation, or shared encoder + per-branch heads |
